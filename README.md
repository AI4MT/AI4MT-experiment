# AI4MT-experiment

This repository contains the results of the experiments with the AI4MT framework. 
Folder [experiments/follow_ups_experiment](./experiments/follow_ups_experiment) contain the follow-up models generated for the case studies (data centers, autonomous vehicles, DFAs)
Folder [experiments/mr_inference_experiment](./experiments/mr_inference_experiment) contain both the gotten programs generated by the MR inference task and the models (and its execution traces) used to analyse their semantic validy.
